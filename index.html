2FA и TG Protect

Ниже кратко о пользе, базовые шаги и видео-инструкции. Страница подходит для ознакомления студентам сервера RPMRolePlay.

Зачем это нужно

Защита от взлома Дополнительная защита даже если пароль стал известен другим людям
2FA (двухфакторная аутентификация)
  • Добавляет второй барьер защиты через приложение.
  • Специальный 6-значный код.
  • Код обновляется каждые 60 секунд.
TG Protect
  • Добавляет второй барьер защиты через Телеграм.
  • Доступ к аккаунту через Телеграм.
  • Вход после нажатия одной кнопки.

Видео: настройка 2FA

Если видео не отображается, откройте напрямую: ссылка на Shorts

Пошаговая инструкция:

  1. Подготовьте аутентификатор. Рекомендуем установить приложение Google Authenticator на телефон, либо использовать расширение для браузера Аутентификатор.
  2. Зайдите в Лобби и введите команду /totp add (или /2fa add).
  3. В чате появится ответ от сервера:
    RPM > Ваш секретный код <длинный код>. Отсканируйте QR код по ссылке
  4. Отсканируйте полученный QR-код через мобильное приложение или введите код вручную.
  5. В приложении сгенерируется 6-значный код. Подтвердите привязку в игре: /2fa confirm XXXXXX.
  6. Готово! Теперь при каждом входе потребуется вводить актуальный код: /2fa code <код>.

Чтобы полностью удалить защиту 2FA, используйте команду: /2fa remove <код>

Видео: настройка TG Protect

Если видео не отображается, откройте напрямую: ссылка на YouTube

Пошаговая инструкция:

  1. Откройте Telegram-бота @rpmroleplay_bot.
  2. Напишите в чат свой игровой ник.
  3. Зайдите в Лобби на сервере и подтвердите привязку.
  4. При входе в игру бот пришлёт уведомление. Нажмите кнопку «Это я», чтобы подтвердить вход.
  5. Теперь ваш аккаунт защищён: никто не сможет войти без вашего подтверждения в Telegram.

Важно: отвязать TG Protect в будущем можно только через нашу техподдержку.
